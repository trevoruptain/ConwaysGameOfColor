Conway's Game of Color

Conway's Game of Life was devised by mathematician John Conway. It's a cellular automaton, which means that the evolution of the game is determined by its initial state. The game consists of a grid of cells, where each cell is either 'alive' or 'dead.' Successive states are determined by the previous state, with each cell on the grid governed by four simple rules:

  • Any live cell with fewer than two live neighbours dies, as if caused by underpopulation
  • Any live cell with two or three live neighbors lives on to the next generation
  • Any live cell with more than three live neighbors dies, as if by overpopulation
  • Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction

This is an interactive version of Conway's Game where a cell's color is determined by its neighbors. The Game of Color is intended to be used in the hero area of a website, with a drawing utility for developers included within the Github repo. Since the grid itself is invisible, the game can overlay a background image. You can interact with the game by inserting predetermined shapes into the grid at any point (check out the buttons at the bottom right). You can also pause or continue the run with the buttons on the left.
